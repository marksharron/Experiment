# The Golden Peacock

A modern static website for **The Golden Peacock**, an Indian restaurant in Brighton and Hove. The site is styled using [Tailwind CSS](https://tailwindcss.com/) and follows a black and gold color scheme.

## Pages
- `index.html` – Home page
- `about.html` – Learn more about the restaurant
- `starters.html` – Starter menu
- `mains.html` – Main courses
- `drinks.html` – Non-alcoholic drinks
- `cocktails.html` – Cocktail menu

Open `index.html` in your browser to explore the site.
